# 文章隐藏测试

![](/blogs/test-conceal/e35665736e0f6829.jpg)

文章隐藏功能其实是测试过两次的，但是效果和我预想的不太一样，不知道是作者的设计巧思还是设计缺陷。

在勾选隐藏选项并保存后，在读者界面确实看不到隐藏文章，并且在导入密钥进入管理员身份时，文章则会显示出来，目前一切正常且是理想状态。但是退出并再次进入管理员身份时，文章则会不翼而飞，并且在GitHub后台也查找不到踪迹。在我问了管理员后，得到的答复是在第二次需要取消勾选，如此看来，这个功能相当于是一次性隐藏的，对我而言，这个功能用来保存草稿就有些鸡肋了，毕竟有时候文章并不是一次两次修改就能发布的。

![](/blogs/test-conceal/a1a42124a8c3dd16.jpg)

今天和一双漂亮的眼睛有了一瞬的接触，在不得见的日子里，那个身影带给我的是无穷的思念，而真正见到了，却是悲伤泛滥成河，有时候我真的无法分辨，你到底是会施加魔法的精灵还是在释放诅咒的魔女。

南有乔木，不可休思；汉有游女，不可求思。汉之广矣，不可泳思；江之永矣，不可方思。

翘翘错薪，言刈其楚；之子于归，言秣其马。汉之广矣，不可泳思；江之永矣，不可方思。

翘翘错薪，言刈其蒌；之子于归，言秣其驹。汉之广矣，不可泳思；江之永矣，不可方思。（**《诗经・国风・周南・汉广》**。）
